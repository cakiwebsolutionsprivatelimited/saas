# Master Memory: SaaS Marketing Website

## Product Overview
This is the public marketing website for a modular Enterprise SaaS business platform. The platform itself consists of multiple internal business modules including CRM, Billing, Inventory, HRMS, Projects, Helpdesk, Super Admin, and Tenant Portals. This website's sole purpose is to market, present, and sell these modules to prospective clients.

## Website Purpose
To act as the public-facing storefront for the SaaS platform. It must clearly communicate the value proposition, showcase the modular applications, present industry-specific solutions, explain pricing structures, and drive lead generation (demo requests and signups).

## Target Audience
B2B decision-makers, enterprise IT directors, business owners, and department heads looking for integrated, scalable, and modular business software solutions.

## Current Pages
* **Home (`/`)**: Main landing page with hero, featured apps, industry solutions, and pricing previews.
* **Apps (`/apps`)**: Detailed catalog and categorization of available SaaS modules.
* **Pricing (`/pricing`)**: Subscription plans and pricing details.
* **Industry Solutions (`/industries`)**: Tailored use-cases for specific business sectors.
* **Request Demo (`/request-demo`)**: Lead capture form for high-touch sales.
* **Login (`/login`)**: Entry point for existing tenants/users (redirects to the actual app).

## Design Philosophy
* **Enterprise SaaS Presentation**: Professional, trustworthy, and scalable.
* **Aesthetics**: Warm neutral palette with restrained Indigo accents.
* **Restrictions**: No glow effects, no neon effects, no decorative gradients.
* **Emphasis**: Achieved through typography, generous spacing, clear hierarchy, and structured layout.

## Current Development Status
Foundation built with React 19, Vite, Tailwind CSS 4, and TypeScript. Routing is configured, and core pages (Home, Apps) are being structured using modular section components.

## Key Website Goals
1. High performance and fast loading.
2. Clean, responsive design across all devices.
3. Strong SEO foundation.
4. High conversion rate via clear CTAs.

## Technology Stack
* **Framework**: React 19 + TypeScript
* **Build Tool**: Vite
* **Routing**: React Router DOM 7
* **Styling**: Tailwind CSS 4 + Class Variance Authority + Tailwind Merge
* **UI Components**: Shadcn UI + Radix UI
* **Icons**: Lucide React
* **Typography**: Geist Font (Fontsource)
